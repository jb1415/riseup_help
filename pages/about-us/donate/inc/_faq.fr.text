h2. Pourquoi n'acceptez-vous pas les virements bancaires sous d'autres devises ?

Un frais important nous est imposé lors de la conversion de devises. De plus, l'envoi direct vers notre compagnie de crédit aux Etats-Unis est souvent compliqué et, souvent, ne fonctionne pas. Lorsqu'un transfert fonctionne, un frais de 13$ par transaction nous est chargé, ce qui peut être dispendieux.

h2. Comment donner en d'autres devises que $US ou EUR ?

Si vous habitez hors des États-Unis, vous pouvez tout d'abord envisager de supporter le collectif informatique local de votre région. Si vous voulez nous envoyer un don, voici quelques options:

# Vous pouvez nous écrire un chèque dans votre monnaie locale et nous l'envoyer à l'adresse plus-haute. Nous acceptons les chèques dans des devises étrangères sans problème, même s'il se peut que la transaction prenne du temps à apparaître sur votre relevé.
# Si vous avez une carte de crédit, vous pouvez utiliser les liens de don électronique plus haut. Vous pouvez également utiliser Paypal, si cette option est diponible dans votre pays.
# Vous pouvez faire un don par "bitcoin":#bitcoin.
# Vous pouvez nous faire parvenir de l'argent sous n'importe quelle devise à notre casier postal. C'est une pratique courante.

h2. Comment modifier ou annuler mes dons récurrents via Paypal ?

Pour ["annuler un don récurent":https://www.paypal.com/us/cgi-bin/webscr?cmd=_help&leafid=2381&answer_id=18037790], suivez ces étapes:

# Ouvrez votre compte Paypal.
# Cliquez sur le sous-onglet *Historique*.
# Choisissez le champs *Abonnements* dans le sous-onglet *Abonnements et contrats* de l'onglet *+ de filtres*.
# Cochez la boîte *À partir de* et reculez la date deux ans en arrière. 
# Cliquez sur *Afficher*.
# Pour voir les détails d'un abonnement spécifique, cliquez sur *Détails* dans la colonne détails.
# Au bas de la page, cliquez sur *Annuler l'abonnement*.
# Sur la page de confirmation, cliquez sur *Annuler l'abonnement* une seconde fois.
